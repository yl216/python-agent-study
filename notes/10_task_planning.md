# 第 10 次学习：简单任务规划

今天的目标是让 Agent 能把一个学习目标拆成可执行步骤。

## 本阶段新增能力

新增 `src/planner.py`，提供：

- `PlanStep`：表示一个计划步骤。
- `TaskPlan`：保存整个计划和当前进度。
- `create_learning_plan()`：根据学习目标生成固定结构的学习计划。

`src/main.py` 新增命令：

```text
/plan <目标>
/plan-show
/plan-next
/plan-reset
```

## 使用示例

```text
/plan 学习 Python 函数
/plan-show
/plan-next
```

## 当前计划结构

每个学习计划默认分成 5 步：

1. 理解核心概念。
2. 运行最小示例。
3. 做一个小改造。
4. 完成练习题。
5. 复盘并记录。

每一步都包含：

- 目标。
- 练习。
- 验收标准。

## 为什么先做固定结构

固定结构不够聪明，但很稳定。

它能保证每个计划都有可执行步骤和验收标准，不会只生成一段看起来很漂亮但无法落地的文字。

后续可以升级为：

- 让模型生成 JSON 计划。
- 本地校验计划结构。
- 自动保存计划到 `notes/`。
- 支持 `skip`、`retry`、`done` 等更细的状态。

## 本阶段验收

- `/plan 学习 Python 函数` 能生成计划。
- `/plan-show` 能查看当前计划。
- `/plan-next` 能推进到下一步。
- `/plan-reset` 能清空计划。
