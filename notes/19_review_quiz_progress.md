# 第 19 次学习：复习、练习和进度

这一阶段把 Agent 从“能管理计划”推进到“能帮助复习”。

新增命令：

```text
/review
/quiz
/progress
```

## /review

读取最近的学习笔记，提取标题、章节和重点条目，生成一份复习材料。

## /quiz

根据最近学习笔记生成简短练习题。

当前版本先做本地生成，不依赖 API。好处是稳定、快、不会因为没有 API Key 而失败。

## /progress

汇总当前学习状态：

- 保存的计划数
- 进行中的计划数
- 已完成的计划数
- 学习笔记数
- 当前计划进度
- 最近笔记

## 你需要理解

这一步的重点不是“生成多聪明的题”，而是学会一个 Agent 产品思路：

```text
已有数据 -> 程序整理 -> 形成上下文 -> 辅助学习
```

后续可以继续升级为：

- 用 DeepSeek 根据复习材料生成更自然的讲解
- 增加 `/grade` 批改用户答案
- 把错题保存到专门的错题笔记
